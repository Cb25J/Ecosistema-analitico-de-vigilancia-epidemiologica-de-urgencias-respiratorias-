# Ejecutable

En el link de google drive adjunto, quedara un archivo .rar con el ejecutable del proyecto en formato .exe y una carpeta con todas las librerias necesarias para su ejecucion, ademas de incluir las bases de datos necesarias para ejecutar el proyecto.

Solamente es necesario descomprimir el archivo .rar y dar doble click al archivo ejecutable .exe

Todas las funciones son operativas, incluso el sistema de actualizacion y descarga del dataset.
